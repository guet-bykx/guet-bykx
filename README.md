## 📖 二、 协会介绍

**八院科协**成立于 **二〇〇一年**。在学院的悉心栽培下，科协二十余年来欣欣向荣，屡创佳绩，为相关行业培养了大量优秀人才。

协会内部氛围团结融洽，各年级、各专业的同学朝夕相处，互相交流、共同进步、彼此成就。在长期的积累下，科协底蕴深厚，有着丰富的学习经验、研究资料，能够帮助成员迅速掌握专业技能、了解行业情况。
> ![大四学长学姐经验分享会|681](https://bykx.nanfault.com/images/Untitled.jpeg)
> ![科协一五级校友马丁学长分享经验|680](https://bykx.nanfault.com/images/Untitled%201.jpeg)

科协是一个**开放包容的集体**，支持成员向自己感兴趣的方向发展。无论喜欢软件或硬件，向往驰骋翱翔的智能车、无人机还是精密的仪器仪表，都能在这里找到志趣相投的同行者。

科协成员们各有所长、团结协作：
*   🏆 **组队竞赛**：在全国大学生电子设计竞赛、全国大学生智能汽车竞赛中屡屡可见来自科协的队伍。
*   🥇 **个人赛事**：在蓝桥杯大赛等个人赛事中也不乏科协成员的身影。

他们学以致用、奋力拼搏，为院校、协会和自身夺得了许多荣誉。

## ⚙️ 三、 协会组成与运转

科协秉承**公正、民主**的思想，各司其职，高效运转：

*   👑 **主席团**：设主席、组织副主席、科技副主席。
*   🏢 **八大部门**：硬件部、软件部、策划部、文宣部、财务部、新媒体部、生活部及维修部。共同完成科协招新、考核、管理与日常维护等工作。
*   👨‍🏫 **指导团队**：三位指导老师，负责提供技术指导与经费支持。

> 💡 **换届机制**：每学年末举办换届大会，由正式成员**投票选举**决定主席、副主席和部长等职位人选。

## 💼 四、 协会工作

在享受学院提供资源的同时，科协也会配合学院完成各项工作：

1.  **承办特色科技活动**
    如学院主办的特色科技活动**“芯梦启航”**。科协会参与活动开幕式与闭幕式的筹办、活动内容规划、人员安排、参与者培训、作品验收及奖品发放等全流程工作。
    > ![科协成员与指导老师在“芯梦启航”闭幕式上合影](https://bykx.nanfault.com/images/Untitled%204.jpeg)*

2.  **对外交流与宣传**
    当学院需要与其他学院、其他学校开展交流活动及进行科技宣传时，科协将指派学生代表参与活动。
    > ![科协代表赴全州市第三中学进行科技宣传](https://bykx.nanfault.com/images/Untitled%205.jpeg)*

3.  **核心任务：专业进阶**
    除上述非日常性工作外，科协成员们最主要的职责仍然是**好好学习，提高专业素养，磨练专业技能**，以图在各类竞赛中取得更好的成绩，为院校谋荣誉，为自身谋发展，而科协会为此提供坚实的支持。

## 🛠️ 五、 协会资源

科协拥有优渥的实验条件与硬软件资源，为成员的成长保驾护航：

*   📍 **实验室位置**：花江慧谷 3#102A实验室。
*   📐 **场地规模**：面积约两百平方米，可容纳四十人进行学习。
*   🔬 **专业仪器**：配备充足的示波器、信号发生器、数控电源及台式数字万用表等各类仪器。
*   🧰 **海量芯片库**：包含大量 TI、ADI、Maxim 等品牌各类芯片，以及 STM32、FPGA、DSP 等各类开发板。

**🎁 正式成员专属福利：**
1.  分配专属的学习工位。
2.  全面开放芯片库、元件库与资料库。
3.  参加各类竞赛时，由科协提供经费购买所需物料。
